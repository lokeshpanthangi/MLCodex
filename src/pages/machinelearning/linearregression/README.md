# Linear Regression from Scratch - Learning Module

## Overview

This folder contains the comprehensive Linear Regression learning module for MLCodex. It's designed to teach complete beginners about Machine Learning starting from the very basics.

## Contents

- `index.tsx` - Main learning module component
- `components/` - Interactive components for the module
  - `InteractiveGraph.tsx` - Interactive linear regression visualization
  - `GradientDescentVisualizer.tsx` - Animated gradient descent visualization
  - `DataPointEditor.tsx` - Allow users to add/edit data points
- `README.md` - This file

## Learning Flow

1. **What is Machine Learning?** - Introduction to ML concepts
2. **How Does ML Work?** - The learning process explained
3. **What is Linear Regression?** - The simplest ML algorithm
4. **The Mathematics** - Understanding the equation y = mx + c
5. **Interactive Playground** - Manipulate slope and intercept to see effects
6. **Cost Function** - Understanding how we measure "wrongness"
7. **Gradient Descent** - How machines learn the best values
8. **Implementation** - Building it from scratch in Python
9. **Practice & Exercises** - Hands-on coding challenges

## Interactive Features

- **Slope & Intercept Sliders** - Adjust m and c values to see line changes
- **Data Point Editor** - Click to add custom data points
- **Gradient Descent Animation** - Watch the algorithm find optimal values
- **Cost Function Visualization** - See how error changes with parameters

## Technologies Used

- React + TypeScript
- Framer Motion for animations
- SVG for custom visualizations
- Tailwind CSS for styling
